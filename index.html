第一篇博客

Posted on 2021-09-22

前言：

总结一下对于一些代码格式的理解

一：示例

首先给大家看一下以下一串代码：
#include<stdio.h>
#include<stdlib.h>
int main()
{
int a, b, c,volume,r;
printf(“请输入长宽高：”);
r = scanf_s(“%d %d %d”, &a, &b, &c);
if (r == 3 && a > 0 && b > 0 && c > 0)
volume = a * b * c;
else printf(“error\n”);
return 0;
}

然后稍微修改一下，如下：
#include<stdio.h>
#include<stdlib.h>

int main()
{
int a, b, c,volume,r;
printf(“请输入长宽高：”);
r = scanf_s(“%d %d %d”, &a, &b, &c);
if (r == 3 && a > 0 && b > 0 && c > 0)
{
volume = a * b * c;
printf(“体积是:%d\n”, volume);
}
else printf(“error\n”);
return 0;
}

二：区别

可以看到两串代码中使用(if)的时候，下面那串在if条件下有用到{}这个符号，而上面这一串代码是不需要用到的。

三：原因

是因为下面的if条件后面有用到其他的关键字，所以需要用到{}。

总结：

在用到函数的时候，如果函数里面有包含关键字，则需要用到{}，如果没有的话就可以不用。

学习汇编1

Posted on 2021-09-22 Edited on 2021-10-21

#今日学习：汇编语言
##重点内容：计算机CPU内使用CS寄存器给的索引地址*16加上IP寄存器给的地址生成一个20位的地址定位到内存的物理地址。
###前言
计算机的外部地址总线的最大寻址能力是20位，而CPU内部只允许通过16位。CPU与外部联系通过三条总线：地址总线，控制总线，数据总线。
###正文
因此，计算机CPU必须通过两段地址相加从而形成一个20位的地址通过地址总线与外部连接，才可以寻址到对应存储形式为20位的内存段。而计算机的CS与IP寄存器默认存入的16进制数为指令，因此通过CS和IP寄存器用地址总线可以连接内存当中的段并运行指令。计算机会默认把CS寄存器中的数乘上16，相当于从16位变成20位再加上IP寄存器中的值变成一个20位数索引到对应的内存地址，得以运行内存里代码。为什么寻址到相应的段的代码会被当作指令运行呢？因为CPU默认通过CS和IP寄存器寻址到的段里的代码就是要执行的指令而不是数据。
###补充
每一次寻址完把指令代码返回给CPU的预处理器以后，CPU中的IP寄存器会自动加上执行过的指令的大小从而达到可以运行下一条命令的效果
###总结
由于刚接触汇编语言，所以也不太了解，这里只是做一个我自以为的小结。计算机就是通过人为的代码输入转换成一个个对应的功能的计算机语言让CPU执行，之所以指令和数据都是一串串由0和1组成的代码，所以计算机必须通过分工来区分数据和指令。
