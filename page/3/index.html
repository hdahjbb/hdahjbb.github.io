学习汇编4

Posted on 2021-11-07

汇编指令：assume segment

引言：

上一篇说到CPU是根据寄存器来判别索引到的地址内存的是数据还是指令，那如何让CPU按照我们的想法去工作呢？这边就涉及到了“伪指令，指令，源程序，程序”的概念。

简介：

源程序：指一个可执行文件内的所有内容。

程序：源程序中可翻译成机器语言的部分指令。

指令：指能够转化为机器语言让计算机工作的程序。

伪指令：指除指令外不能翻译为机器语言的所有程序。

说白了就是编译器编译完源程序，将其中的指令转化为二进制形成一个程序。

工作原理：‘

在编译器中使用assume：cs，告诉编译器以下代码直至ends的段为指令段。XXX segment为这个段的命名，同时还有一个作用，编译器会将XXX变为一个地址当作这个段的头地址。直到ends出现表明这个代码段结束。最后以end结束，把控制权交回给操作系统。

学习汇编3

Posted on 2021-10-31 Edited on 2021-11-21

前言：

今天初步学习了栈，在这里记录一下对栈的初步理解。

我对栈的概念的理解：

栈是一个人为定义的特殊的内存空间，它与其他内存段不同的主要在于FILO（first in last out），就是先推入栈的值会最后出来，就如同叠书本一样，垫底的是放进去的，最后拿出来的就是那本垫底的，这就是栈与其他内存空间的不同。

与栈相关的指令：

由于是初步学习，所以就只知道push和pop。push就是把数据推入栈，pop就是把栈里的数据拿踩出来。值得注意的是，在push的时候，CPU不是通过CS寄存器寻址，而是通过SS寄存器，与其他段地址寄存器一样，SS也不能直接输入数据，必须通过通用寄存器来交接。
以上就是我对栈的初步认识。

一些其他的知识：

当我们格式化硬盘的时候，其实数据并没有抹去，而是改变了CPU的索引，直到下一步操作，新数据覆盖了原有的数据，硬盘里的旧数据才会消失。

学习汇编1

Posted on 2021-09-22 Edited on 2021-11-21

今日学习：汇编语言

重点内容：

计算机CPU内使用CS寄存器给的索引地址*16加上IP寄存器给的地址生成一个20位的地址定位到内存的物理地址。

前言：

计算机的外部地址总线的最大寻址能力是20位，而CPU内部只允许通过16位。CPU与外部联系通过三条总线：地址总线，控制总线，数据总线。

正文：

因此，计算机CPU必须通过两段地址相加从而形成一个20位的地址通过地址总线与外部连接，才可以寻址到对应存储形式为20位的内存段。而计算机的CS与IP寄存器默认存入的16进制数为指令，因此通过CS和IP寄存器用地址总线可以连接内存当中的段并运行指令。计算机会默认把CS寄存器中的数乘上16，相当于从16位变成20位再加上IP寄存器中的值变成一个20位数索引到对应的内存地址，得以运行内存里代码。为什么寻址到相应的段的代码会被当作指令运行呢？因为CPU默认通过CS和IP寄存器寻址到的段里的代码就是要执行的指令而不是数据。

补充：

每一次寻址完把指令代码返回给CPU的预处理器以后，CPU中的IP寄存器会自动加上执行过的指令的大小从而达到可以运行下一条命令的效果

总结：

由于刚接触汇编语言，所以也不太了解，这里只是做一个我自以为的小结。计算机就是通过人为的代码输入转换成一个个对应的功能的计算机语言让CPU执行，之所以指令和数据都是一串串由0和1组成的代码，所以计算机必须通过分工来区分数据和指令。

第一篇博客

Posted on 2021-09-22

前言：

总结一下对于一些代码格式的理解

一：示例

首先给大家看一下以下一串代码：
#include<stdio.h>
#include<stdlib.h>
int main()
{
int a, b, c,volume,r;
printf(“请输入长宽高：”);
r = scanf_s(“%d %d %d”, &a, &b, &c);
if (r == 3 && a > 0 && b > 0 && c > 0)
volume = a * b * c;
else printf(“error\n”);
return 0;
}

然后稍微修改一下，如下：
#include<stdio.h>
#include<stdlib.h>

int main()
{
int a, b, c,volume,r;
printf(“请输入长宽高：”);
r = scanf_s(“%d %d %d”, &a, &b, &c);
if (r == 3 && a > 0 && b > 0 && c > 0)
{
volume = a * b * c;
printf(“体积是:%d\n”, volume);
}
else printf(“error\n”);
return 0;
}

二：区别

可以看到两串代码中使用(if)的时候，下面那串在if条件下有用到{}这个符号，而上面这一串代码是不需要用到的。

三：原因

是因为下面的if条件后面有用到其他的关键字，所以需要用到{}。

总结：

在用到函数的时候，如果函数里面有包含关键字，则需要用到{}，如果没有的话就可以不用。
