第一篇博客

前言：

总结一下对于一些代码格式的理解

一：示例

首先给大家看一下以下一串代码：
#include<stdio.h>
#include<stdlib.h>
int main()
{
int a, b, c,volume,r;
printf(“请输入长宽高：”);
r = scanf_s(“%d %d %d”, &a, &b, &c);
if (r == 3 && a > 0 && b > 0 && c > 0)
volume = a * b * c;
else printf(“error\n”);
return 0;
}

然后稍微修改一下，如下：
#include<stdio.h>
#include<stdlib.h>

int main()
{
int a, b, c,volume,r;
printf(“请输入长宽高：”);
r = scanf_s(“%d %d %d”, &a, &b, &c);
if (r == 3 && a > 0 && b > 0 && c > 0)
{
volume = a * b * c;
printf(“体积是:%d\n”, volume);
}
else printf(“error\n”);
return 0;
}

二：区别

可以看到两串代码中使用(if)的时候，下面那串在if条件下有用到{}这个符号，而上面这一串代码是不需要用到的。

三：原因

是因为下面的if条件后面有用到其他的关键字，所以需要用到{}。

总结：

在用到函数的时候，如果函数里面有包含关键字，则需要用到{}，如果没有的话就可以不用。
